Welcome to the official repository for **[abinvinod.in](https://abinvinod.in)**, the professional portfolio and digital marketing agency platform for **Abin Vinod** — the leading Digital Marketer & SEO Expert in Kerala, India.

This platform is engineered for blazing-fast performance, premium aesthetics, and search engine dominance. It showcases advanced local SEO landing pages, structured schemas, a blogging platform, and high-conversion UI/UX components.

---

## ✨ Features

### 🌟 Premium UI/UX & Aesthetics
* **Glassmorphic Theme**: Dark-mode inspired palette featuring smooth gradients, subtle glowing background orbs, and glassmorphic panels.
* **Custom Cursor**: Dynamic dual-dot custom cursor that morphs and interacts with hoverable elements.
* **Micro-interactions**: Magnetic button effects, text-reveal animations, and scroll-triggered fade-ins powered by modern CSS and lightweight vanilla JS.
* **Responsive Design**: Mobile-first architecture that scales seamlessly from small mobile screens to large desktop monitors.

### 📈 Advanced SEO & Performance
* **Structured Data (JSON-LD)**: Rich schema integration including `Person` schema, `FAQPage` schema, and article metadata to maximize rich snippet visibility on search engine results pages (SERPs).
* **Multi-District Local SEO Funnel**: Custom landing pages targeted specifically to all 14 districts of Kerala (e.g., Kochi, Kozhikode, Trivandrum) for hyper-local search intent capturing.
* **Core Web Vitals Optimized**: Zero heavy framework overhead; built entirely using semantic HTML5, pure CSS3, and native JavaScript to ensure instant load times.
* **Dynamic Sitemap & Robots**: Fully configured `sitemap.xml` and `robots.txt` optimized for search crawlers.

### 📝 Blog System
* Optimized blog articles targeting high-intent keywords like digital marketing guides and SEO tips for small businesses in Kerala.
* Clean semantic layout designed to maximize reading comfort and user session duration.

## 🚀 Getting Started

Since this project is built entirely on native web technologies, there are no compilation steps or heavy dependencies required.

### Local Development / Preview

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dingan2218/abinvinod.git
   cd abinvinod
   ```

2. **Serve the project:**
   You can open `index.html` directly in any web browser, or use a lightweight local server for hot-reloading (highly recommended for custom cursors and smooth scripts):
   * **VS Code:** Use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension.
   * **Python:** Run `python -m http.server 8000` and visit `http://localhost:8000`.
   * **Node.js (npx):** Run `npx serve` or `npx live-server`.
